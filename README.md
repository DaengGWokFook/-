# 漢字文化圈語言文字輸入方案

## 簡介：
製作漢字相關語音字典、辭典及輸入方案，目的只爲保存各地區語言、文字與文化。
開發內容包含：華北語系、吳語系，閩語系、越語系（粵語）、贛語、湘語、晉語、徽語、平語（平話）、客語、高麗語（朝鮮語和韓國語）、日本語以及古代語音等。
希望得到各位的幫助，提供資料和指出錯誤。

## 翻譯：
因爲此網站的資料夾只能用羅馬字母與部分符號命名，所以無法用漢字命名，因此給諸位帶來不便，請見諒。
